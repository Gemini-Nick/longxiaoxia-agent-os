# chan.AI Landing Page

This directory contains the static GitHub Pages draft for chan.AI.

Open locally:

```bash
open docs/landing/chan-ai/index.html
```

Publish through GitHub Pages:

1. Open the repository settings on GitHub.
2. Set Pages source to `GitHub Actions`.
3. Run or push to trigger `.github/workflows/pages.yml`.
4. The page will be available at `/landing/chan-ai/`.

The landing page is intentionally static. It should explain the product before it depends on a deployment pipeline.
